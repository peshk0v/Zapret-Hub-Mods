# Zapret Hub 1.4.3

## Исправлено
- Исправлено обновление компонента Zapret: апдейтер теперь принимает оба формата upstream-архива, корректно обновляет оригинальный `runtime/zapret-discord-youtube`, а не `merged_runtime`, и после успешного обновления пересобирает видимый merged runtime.
- Встроенный Zapret обновлён до upstream `1.9.8`.
- Снижена вероятность ошибки Easy Anti-Cheat вида “WinDivert64.sys loaded, but its file is missing”: Zapret Hub больше не удаляет active runtime-папку, из которой мог быть загружен WinDivert-драйвер.
- Исправлена нестабильная загрузка списка конфигураций Zapret на странице “Компоненты”: список теперь догружается через backend payload и не требует повторного входа на страницу.
- В редактор файлов добавлен отдельный режим “General” для редактирования доступных `.bat/.cmd` конфигураций Zapret через существующий backend-save путь.

## Добавлено
- В настройках TG WS Proxy добавлены параметры DC -> IP, режим для фото/видео Telegram, Cloudflare fallback, приоритет Cloudflare, пользовательский CF-домен, Fake TLS domain, размер буфера и pool size.
- На странице “Компоненты” добавлены кнопки “Источник” для Zapret и TG WS Proxy, открывающие оригинальные GitHub-репозитории Flowseal.

## Улучшено
- Обновление TG WS Proxy и Zapret стало устойчивее к сетевым сбоям: добавлены повторные попытки скачивания и более понятная диагностика ошибок архива.
- GitHub Actions и локальные Nuitka-скрипты подготовлены под версию `1.4.3`.
- Окно настроек стало компактнее: настройки сгруппированы в скроллируемые блоки “Приложение”, “Zapret” и “TG WS Proxy”.
- Настройки стали выше, получили плавный скролл и более спокойные outline-границы секций без акцентной синей подсветки.
- Иконка “Источник” приведена к стилю остальных action-кнопок и автоматически подстраивается под светлые и тёмные темы.
